# Service Terminology

## Naming Policy

- Use Korean-first labels in all user-facing UI copy.
- Keep brand and feature names in fixed forms; do not mix variants.
- Use English snake/kebab identifiers for code, API, and analytics fields.

## Canonical Terms

| Type | Canonical Label | Internal Identifier | Notes |
| --- | --- | --- | --- |
| Brand | Whale Radar | `whale-radar` | Product/service brand name |
| Product | 기관 보유공시 분석 대시보드 | `dashboard-shell` | Main dashboard title |
| Tab | 마켓 허브 | `market` | Market-wide aggregate view |
| Tab | 운용사 동향 | `whale` | Manager-level holdings and action view |

## Deprecated Terms

- `WhaleInsight Pro` -> use `Whale Radar`
- `Whale Insider` -> use `운용사 동향`
